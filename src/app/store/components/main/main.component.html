Данный сайт представляет собой интернет-магазин. Прототипом приложения можно считать сайт электронного магазина 21 Век. Данный проект реализован с помощью фраемвока Angular. Приложение реализовано по принципу SPA.


Реализован переход по ссылкам внутри сайта, так же добавлено страница 404, для исключения ошибок.


При загрузке преложения идет определение местоположения пользователя и отображение в строке состояния в header, при неточном определении можно выбрать точное местоположение с помощью pop-up меню(Реализованы все города Беларуси).


На главной страницы присутствует слайдер с акциями и свапер с популярными товарами.


Реализован поиск, авторизация, навигация по категориям и подкатегориям с иерархией. Возможно осуществить сортировку по таварам заданной катгории.


Работа с товарами осуществляется как на отдельной странице так и со страницы каегорий.


Из меню доступны: Избранные товары, Корзина и Лист ожидания.
